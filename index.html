Welkom op mijn website,

Mijn naam is Owen ik ben 18 jaar oud en heb ervaring met HTML, CSS, Javascript, PHP, Linux en FreeBSD.
Deze website is gebouwd om in groot detail te laten zien wat ik kan en waar ik ervaring mee heb,
ook bevat deze website mijn hobby’s en cv .pdf bestand om te downloaden of direct te bekijken als je op een desktop systeem bent.
Ook is er als voorbeeld op de hobby’s pagina een volledig HTML, CSS en Javascript muziek speler met pauze, skip en rewind knoppen om door de muziek te bladeren of opnieuw af te spelen.
Zoals je kan lezen in de portfolio pagina ben ik goed in Engels en heb ik zelfs meegemaakt dat mensen dachten dat ik Amerikaans,
ik heb bij bedrijven gewerkt die niks te maken hebben met ICT maar toch heb ik bij Jumbo en Aldi een jaar gewerkt en heb ik nu een contractverlenging gekregen bij Makro Den Bosch.
Mijn hobby’s zijn samen met werk best dicht bij elkaar en daardoor ben ik op het nog best wel breed vlak van ICT handig met verschillende tools of software waardoor ik minder zoek en sneller kan werken met tools die ik zelf fijn vind om te gebruiken.
Ik gebruik het liefst voor het programmeren een bewerkte versie van Visual Studio Code genaamd VSCodium die net iets sneller werkt en geen Microsoft Onderdelen gebruikt zoals een account of automatische Cloud synchronisatie, die de app vertragen.
